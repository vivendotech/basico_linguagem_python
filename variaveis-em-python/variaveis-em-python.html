Variáveis em Python

Em uma linguagem de programação além de poder definir funções você pode definir variáveis. Para criar uma variável em Python basta escrever um nome que não tenha sido ainda definido, utilizar o comando de atribuição ( = ) e atribuir um valor

. Lembre-se que esse nome não pode começar por um numero (e.g. 1nome) ou ser só um numero (e.g. 1). Você pode ver um exemplo de como criar variáveis em Python no código 1.

# Código 1 - Criando variáveis em Python mensagem = 'Ola mundo' numero = 520 numeroPi = 3.141592

Python atribui dinamicamente os tipos dos dados

No Python como em outras linguagens você não precisa previamente definir estaticamente o tipo de variável. No Python, as variáveis nunca são explicitamente definidas/escritas. O Python descobre qual é o tipo de variável no momento que é atribuído um valor a ela e, a monitora de forma interna. [4] Por isso dissemos que o Python é uma linguagem com definições dinâmicas. Isso é de certa forma condizente com seu design voltado para ajudar o programador.

As variáveis no Python também são fortemente definidas. Ou seja, o tipo da variável é sempre reforçado. Se você tem uma variável inteira, você não pode tratar ela como uma variável do tipo string sem ter uma conversão explicita antes. Se ela fosse uma linguagem com variáveis fracamente definidas os tipos das variáveis poderiam ser ignorados. Uma linguagem que possui variáveis fracamente definidas por exemplo é a VBScript [4]. Em VBScript você pode concatenar uma string (’12’) com um numero do tipo inteiro ( 3 ) para ter a string (‘123’) sem precisar explicitar nenhuma conversão antes.

Então no Python é uma linguagem com definições de tipos de forma dinâmica porque você não precisa explicitamente definir um tipo antes de criar uma váriavel. E os tipos de váriaveis são fortes já que depois que foi atribuído um tipo a esse dado esse tipo de dado importa[4].

Identificando o tipo das variáveis

Iremos identificar os tipos dos dados que nós criamos. Para isso vamos usar a função “print” e a função “type”. Essas funções já vem definidas/criadas pela biblioteca padrão do Python. A função “print” serve para imprimir o texto que entra como argumento no prompt de comando. E a função “type” serve para retornar que tipo é a variável que entrou nela pelo seu argumento. Vale mencionar que o Python é uma linguagem de programação de propósito geral e que já vem com as baterias inclusas. Isso significa que você já consegue fazer uma série de programas somente com a biblioteca padrão e ela serve para fazer um monte de aplicação diferentes.

# código 2 - Imprimindo o tipo das viráveis em Python print(type(message)) # str print(type(n)) # int print(type(pi)) # float

Ao observar o código 3 percebemos que cada uma das variáveis possuí um tipo primitivo diferente. A variável “mensagem” é do tipo string (texto), a variável “n” é do tipo inteiro, e a variável pi é do tipo ponto flutuante. O Python internamente atribuiu esses tipos a essas variáveis. O Python tem vários tipos de variáveis construídas em sua biblioteca padrão, que iremos discutir ao longo de outros tutorias.
